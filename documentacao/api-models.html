Item

Classe abstrata que representa um item genérico no sistema. Serve como classe base para diferentes tipos de itens como Produtos e Materiais.

Propriedades

Id int

Identificador único do item.

UserId string?

Identificador do usuário proprietário/criador do item.

CategoryId int?

Identificador da categoria à qual o item pertence.

Name string

Nome do item. Obrigatório, máximo de 100 caracteres.

Reference string

Referência/código do item. Obrigatório, máximo de 50 caracteres.

Description string

Descrição detalhada do item. Obrigatório, máximo de 500 caracteres.

Price float

Preço unitário do item. Obrigatório, deve ser maior que zero.

Stock float

Quantidade em estoque do item. Valor padrão é zero.

Unit string

Unidade de medida do item (ex: kg, unidade, m²). Obrigatório, máximo de 20 caracteres.

CreatedAt DateTime

Data de criação do item, gerada automaticamente.

Category Category?

Categoria à qual o item pertence. Relacionamento com a entidade Category.

User User?

Usuário proprietário/criador do item. Relacionamento com a entidade User.

Images ICollection<ItemImage>

Coleção de imagens associadas ao item.

Product

Representa um produto no sistema. Herda da classe abstrata Item, estendendo-a com propriedades específicas de produtos.

Propriedades

ProductMaterials ICollection<ProductMaterial>

Coleção de materiais associados ao produto. Relacionamento com a entidade ProductMaterial que conecta produtos e materiais.

Tags ICollection<Tag>

Coleção de tags associadas ao produto. Permite categorizar e filtrar produtos por características específicas.

Order

Representa um pedido no sistema. Armazena informações sobre pedidos feitos pelos usuários, incluindo os itens solicitados e o status atual.

Propriedades

Id int

Identificador único do pedido.

UserId string?

Identificador do usuário que realizou o pedido.

Status OrderStatus

Status atual do pedido (Pendente, Confirmado, Enviado, Entregue, Cancelado).

User User?

Usuário que realizou o pedido. Relacionamento com a entidade User.

CreatedAt DateTime

Data de criação do pedido, gerada automaticamente.

OrderItems ICollection<OrderItem>

Coleção de itens incluídos no pedido.

Métodos

GetTotal() float

Calcula o valor total do pedido somando os valores de todos os itens incluídos.

OrderItem

Representa um item específico dentro de um pedido. Armazena informações sobre a quantidade e o item solicitado.

Propriedades

Id int

Identificador único do item do pedido.

Quantity float

Quantidade solicitada do item.

OrderId int?

Identificador do pedido ao qual este item pertence.

Order Order?

Pedido ao qual este item pertence. Relacionamento com a entidade Order.

ItemId int?

Identificador do item (produto ou material) que foi pedido.

Item Item

Item (produto ou material) que foi pedido. Relacionamento com a entidade Item.

Métodos

GetTotal() float

Calcula o valor total deste item no pedido (preço unitário × quantidade).

Category

Representa uma categoria para produtos e materiais no sistema. Permite organizar itens em grupos lógicos para facilitar a navegação e filtragem.

Propriedades

Id int

Identificador único da categoria.

Reference string

Código de referência da categoria. Obrigatório, máximo de 50 caracteres.

UserId string?

Identificador do usuário que criou a categoria.

Name string

Nome da categoria. Obrigatório, máximo de 100 caracteres.

CreatedAt DateTime

Data de criação da categoria, gerada automaticamente.

Products ICollection<Product>

Coleção de produtos associados a esta categoria.

Materials ICollection<Material>

Coleção de materiais associados a esta categoria.

User User?

Usuário que criou a categoria. Relacionamento com a entidade User.

User

Representa um usuário no sistema. Estende a classe IdentityUser do ASP.NET Identity para incluir informações adicionais específicas do sistema.

Propriedades

FullName string

Nome completo do usuário. Campo obrigatório.

Street string?

Endereço (rua) do usuário.

StreetNumber string?

Número da residência/estabelecimento do usuário.

City string?

Cidade/localidade do usuário.

ZipCode string?

Código postal do endereço do usuário.

Website string?

Website pessoal ou da empresa do usuário.

Description string?

Descrição ou biografia do usuário.

AcceptedByAdmin bool

Indica se o usuário foi aceito/aprovado por um administrador. Valor padrão é false (não aprovado).

CreatedAt DateTime

Data de criação da conta do usuário, gerada automaticamente.

Role UserRole

Papel/função do usuário no sistema (Artesão, Cliente, Admin). Este campo não é persistido diretamente no banco de dados.

Products ICollection<Product>

Coleção de produtos associados a este usuário (criados ou gerenciados por ele).

Materials ICollection<Material>

Coleção de materiais associados a este usuário (criados ou gerenciados por ele).

FavoriteMaterials ICollection<UserMaterialFavorite>

Coleção de materiais marcados como favoritos por este usuário.

Orders ICollection<Order>

Coleção de pedidos feitos por este usuário.
